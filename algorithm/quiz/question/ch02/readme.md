# 가위바위보

코딩을 좋아하는 지으니는 친구 개발자들과 함께 봇을 이용한 가위바위보 게임을 하려고 합니다. 가위바위보를 판단할 수 있는 심판관을 만들어주세요.


```
[사전지식]
심판관은 봇에게 이름과, 매 판마다 바위는 0, 가위는 1, 보는 2로 전달받습니다.
입력값은 String이거나 Number입니다.

매번 승자는 최소 1점의 점수를 얻으며, 연승 시 승수만큼의 점수를 더 획득합니다.
1번 이김 = 1점
2번 연속 이김 = 2점
3번 연속 이김 = 3점
....

지거나 비길 경우 연속 콤보는 0이 됩니다.

ex)
A bot
win, win, win, lose, win
1, 2, 3, 0, 1 
총 점수 = 7점

B bot
lose, lose, lose, win, lose
0, 0, 0, 1, 0
총 점수 = 1점


최종 결과로 전적을 알려주세요.

[봇]
봇들은 이름과 개별 실행함수를 갖고 있습니다.
bot {
  name: 'bot_name',
  handler: () => {}
}

bot의 handler를 매번 독립시행 후, 결과를 다음과 같이 도출해주세요.

[응답]
result {
  [bot1_name]: 최종 점수,
  [bot2_name]: 최종 점수,
  total: 전체 라운드 수
}

```

```js
function inspector (bot1, bot2, round) {
  /**
   * 원하는 코드를 작성해주세요. 
   *
   **/

  return {
    [bot1.name]: 0,
    [bot2.name]: 0,
    total: round
  }
}
```
